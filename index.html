🎓 Queen's Student Card Tutorial

Learn to create your own personalized university student website

Welcome to the Tutorial!

In this tutorial, you'll learn how to create a beautiful, personalized student card website that showcases your university experience. Perfect for portfolios, sharing with friends, or just having fun with web development!

📚 Tutorial Steps

Step 1: Set Up Your HTML Structure

We'll start by creating the basic HTML structure for your student card.

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My Queen's Student Card</title>
    <link rel="stylesheet" href="student-card.css">
</head>
<body>
    <div class="student-card">
        <!-- Your content will go here -->
    </div>
</body>
</html>

Step 2: Add Your Personal Information

Create the header section with your photo and basic info.

<div class="card-header">
    <div class="photo-section">
        <img src="your-photo.jpg" alt="Your Name" class="profile-photo">
    </div>
    <div class="basic-info">
        <h1>Your Name</h1>
        <p class="student-id">Student ID: 20XXXXXX</p>
        <p class="program">Your Program</p>
        <p class="year">Year X</p>
    </div>
</div>

Step 3: Add Academic Information

Include your academic details and achievements.

<div class="academic-info">
    <h2>📊 Academic Information</h2>
    <div class="info-grid">
        <div class="info-item">
            <span class="label">Faculty:</span>
            <span class="value">Your Faculty</span>
        </div>
        <div class="info-item">
            <span class="label">GPA:</span>
            <span class="value">X.XX</span>
        </div>
    </div>
</div>

Step 4: Style Your Card with CSS

Now let's make it look great! Create a CSS file to style your student card.

.student-card {
    max-width: 600px;
    margin: 2rem auto;
    background: linear-gradient(135deg, #1e3a8a, #3b82f6);
    color: white;
    border-radius: 20px;
    padding: 2rem;
    box-shadow: 0 10px 30px rgba(0,0,0,0.2);
}

.profile-photo {
    width: 120px;
    height: 120px;
    border-radius: 50%;
    object-fit: cover;
    border: 4px solid white;
}

Step 5: Add Interactive Elements

Make your card interactive with hover effects and animations.

.student-card:hover {
    transform: translateY(-5px);
    box-shadow: 0 15px 40px rgba(0,0,0,0.3);
    transition: all 0.3s ease;
}

.info-item:hover {
    background: rgba(255,255,255,0.1);
    border-radius: 8px;
    padding: 0.5rem;
    transition: all 0.2s ease;
}

Step 6: Customize and Personalize

Add your own touches! Consider adding:

  • 🎨 Your favorite colors or Queen's colors
  • 🏆 Achievements and awards
  • 📚 Favorite courses or professors
  • 🎯 Career goals or interests
  • 📱 Social media links
  • 🎵 Fun facts about yourself
